Core Capabilities

Everything you need to ship

Battle-tested components that handle infrastructure plumbing so you can focus on business logic.

Structured Logging

Zerolog-based logging with console and JSON (Logstash/ELK) output modes, configurable levels, and custom writers.

Health Checks

Kubernetes-compatible readiness and liveness endpoints with custom check function support for both Gin and Chi.

Prometheus Metrics

Automatic /metrics endpoint registration. Expose runtime and application metrics in Prometheus format.

OAuth2 + PKCE

Production OAuth2 flows for Google and Apple with PKCE support, token exchange, and user profile retrieval.

HTTP Middleware

Request logging with path exclusions, CORS configuration, panic recovery, and request timeout enforcement.

File Uploads

Configurable categories with MIME type validation, magic-byte verification, UUID filenames, and path traversal protection.

Quick Start

Up and running in seconds

Set up a production-ready HTTP server with health checks, metrics, and structured logging.

Chi Router Setup

A complete server with health checks, Prometheus metrics, structured logging, middleware, and graceful shutdown — in under 30 lines.

Graceful shutdown with configurable timeout
Kubernetes-ready health endpoints
Prometheus metrics out of the box
Production middleware stack included
main.go 
package main

import (
    "github.com/PhilipKram/gms-foundation/pkg/server/chi"
    "github.com/PhilipKram/gms-foundation/pkg/healthcheck"
    "github.com/PhilipKram/gms-foundation/pkg/prometheus"
    "github.com/PhilipKram/gms-foundation/pkg/logger"
    "github.com/rs/zerolog"
)

func main() {
    logger.SetupLogger(logger.ConfigSchema{
        Level: int8(zerolog.InfoLevel),
    })

    srv, router := chi.Setup(chi.ConfigSchema{
        Port:      "8080",
        AccessLog: true,
    })

    healthcheck.RegisterChi(router)
    prometheus.RegisterChi(router)

    router.Get("/api/hello", yourHandler)

    chi.Start(srv)
}
